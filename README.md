# AdventOfCode 2024 Hogent/TIAO style

## Introduction

Streams make java bearable.  I find AoC excellent for exploring new programming languages.
[about AdventOfCode](https://adventofcode.com/2024/about)

- 25 days, each day a new puzzle, each puzzle has 2 parts


## Get Started

We start with a solved Day1.

Each day has an input.txt file which we save in `src/main/resources` as dayXX.txt, ie day01.txt.

Write a new class DayXX in `src/main/java/domein`, ie `src/main/java/domein/Day01.java`.

That's it.

Run from javafx:run or using mvn run
